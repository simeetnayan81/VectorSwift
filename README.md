# VectorSwift

Swift library for vector storage and similarity search.

## Requirements

- macOS with Xcode (full Xcode app, not only Command Line Tools)
- Swift 5.10+

```bash
sudo xcode-select -s /Applications/Xcode.app/Contents/Developer
```

## Build & test

```bash
swift build
swift test
```

## Use

```swift
import VectorSwift
```

The public product is the `VectorSwift` library target.
